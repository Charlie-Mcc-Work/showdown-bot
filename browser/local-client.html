Pokemon Showdown

Play Against Your Trained Bot

1 Start the Pokemon Showdown Server

Open a terminal and run:

cd ~/pokemon-showdown && node pokemon-showdown start --no-security

2 Start the Trained Bot

Open another terminal and run:

cd ~/Documents/Programming/Charlie-Mcc-Work/showdown-bot
.venv-rocm/bin/python scripts/play.py

The bot will wait for challenges as "TrainedBot".

3 Start the Client HTTP Server

Open another terminal and run:

cd ~/pokemon-showdown-client && npx http-server -p 8080 -c-1

4 Open the Game Client

Click the link below to open the Pokemon Showdown testclient:

Open Pokemon Showdown Client

5 Login (Important!)

The testclient will show "verification signature invalid" when you try to pick a name. To fix this, use the browser console:

  1. Press F12 to open Developer Tools
  2. Go to the Console tab
  3. Paste this command and press Enter:
app.socket.send('|/trn Player,0,')

Alternative: Drag this bookmarklet to your bookmarks bar:

PS Local Login

Then click it while on the testclient page to login.

6 Challenge the Bot

  1. Once logged in, click "Find a user" in the right panel
  2. Search for: TrainedBot
  3. Click "Challenge"
  4. Select format: gen9randombattle
  5. Click "Challenge" to start the battle!
